# pinned
📌 github pinned repos, in rust

# ❓ how to use
just go to the path of the user you want to get the pinned repos for! e.g.
```
https://pinnd.terabyteis.me/terabyte3
```

# installation
```sh
$ git clone https://github.com/terabyte3/pinned
$ cd pinned
$ cargo run
```
